# Resume Tailor

Resume Tailor helps job seekers quickly align their resume content with a specific job description.

## Live demo

[Live demo](https://your-vercel-url-here)

## Features

- Resume paste or file upload (PDF and DOCX)
- Job description input
- AI-powered per-bullet rewrite suggestions
- Keyword gap analysis
- Saved resumes via localStorage
- Dark mode default with toggle

## Tech stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Google Gemini API (or AI provider of your choice)
- mammoth.js
- pdf.js

## Getting started

1. Clone the repository:
   `git clone <your-repo-url>`
2. Move into the project folder:
   `cd resume-tailor`
3. Create a `.env` file from `.env.example`.
4. Add your AI API key to `.env`.
5. Install dependencies:
   `npm install`
6. Start the dev server:
   `npm run dev`
